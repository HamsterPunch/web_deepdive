10.7 프로퍼티 동적 생성

존재하지 않는 프로퍼티에 값을 할당하면 프로퍼티가 동적으로 생성되어 추가되고 프로퍼티 값이 할당된다. 

                    
                        var person = {
                            name: 'Lee'
                        };

                        // person 객체에는 age 프로퍼티가 존재하지 않는다.
                        person.age = 20;

                        console.log(person); // {name: 'Lee', age: 20}
