8.5 continue 문

continue 문은 반복문의 코드 블록 실행을 현 지점에서 중단하고 반복문의 증감식으로 실행 흐름을 이동시킨다. break 문처럼 반복문을 탈출하지는 않는다. 

                    
                        var string = 'Hello World.';
                        var search = 'l';
                        var count = 0;

                        for (var i = 0; i < string.length; i++) {
                            if (string[i] !== search) continue;
                            count++;
                        }

                        console.log(count); // 3
