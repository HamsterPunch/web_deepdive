30.2 Date 메서드

Date.now 메서드는 1970년 1월 1일 00:00:00(UTC)을 기점으로 현재 시간까지 경과한 밀리초를 숫자로 반환한다. 

                        
                            const now = Date.now();
                            new Date(now); // Mon Jul 06 2020 01:03:18 GMT+0900 (대한민국 표준시)

Date.parse 메서드는 1970년 1월 1일 00:00:00(UTC)을 기점으로 인수로 전달된 지정 시간까지의 밀리초를 숫자로 반환한다. 

                        
                            // UTC
                            Date.parse('Jan 2, 1970 00:00:00 UTC'); // 86400000

                            // KST
                            Date.parse('Jan 2, 1970 09:00:00'); // 86400000

                            // KST
                            Date.parse('1970/01/02/09:00:00'); // 86400000
